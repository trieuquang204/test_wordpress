== Description ==

** THIS PLUGIN WORK ONLY WITH MISTERCORPORATE WORDPRESS THEME **

** GET THE FREE THEME! **
You can download Mistercorporate WordPress theme in the official WordPress repository <a href="https://wordpress.org/themes/mistercorporate/" target="_blank">https://wordpress.org/themes/mistercorporate/</a>

This plugin allow you to add more features to this theme.
Activate this plugin and then go to Appearance -> Customize.
In "Mistercorporate Options" you can find:
- Quote section
- Try It section
- Newsletter section
- Adv section
- Price section
- Partner section
- Map section
- Contact section

Create and display this sections is very easy!
See the documentation here: <a href="http://mistercorporate.nsthemes.com/documentation/" target="_blank">http://mistercorporate.nsthemes.com/documentation/</a>


For more information our visit official page <a href="http://www.nsthemes.com/" target="_blank">NsThemes Marketplace</a>.

*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at <a href="mailto:report@nsthemes.com">report@nsthemes.com</a>.*


== Installation ==

1. Upload the entire 'ns-mistercorporate' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Appearance -> Customize'.
